# scoop-vnxt

Scoop bucket for [vnxt](https://vnxt.dev) — a lightweight CLI tool for automated semantic version bumping with changelog generation and git integration.

## Install

```powershell
scoop bucket add vnxt https://github.com/n-orrow/scoop-vnxt
scoop install vnxt
```

## Usage

```powershell
vx -m "feat: add new feature"
```

See [vnxt.dev](https://vnxt.dev) for full documentation.

## Requirements

- [Node.js](https://nodejs.org) (install via `scoop install nodejs` if needed)
